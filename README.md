# Conductor

Local project and task manager with MCP integration — manage your projects and tasks from a desktop UI or HTTP API, with full AI assistant integration.

## Features

- **Project & task management** — create projects, break them into tasks, track status and progress
- **MCP server** — 13 tools for AI assistant integration (works with Claude, NeoCode, AgentFactory, etc.)
- **HTTP API** — full REST API for integration with other tools
- **AgentFactory integration** — trigger pipelines, track job status, log activity
- **Service token auth** — secure inter-service communication between tools
- **Activity log** — track everything AI agents do across your projects

## MCP Tools

`cnd_list_projects`, `cnd_get_project`, `cnd_create_project`, `cnd_list_tasks`, `cnd_get_task`, `cnd_create_task`, `cnd_update_task`, `cnd_log_activity`, `cnd_get_activity`, `cnd_trigger_pipeline`, `cnd_get_af_job`, and more.
